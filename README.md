# Chaabane Portfolio (React + Vite)

A creative, SEO-friendly personal portfolio for **Chaabane Boussadia**, focused on software architecture and engineering impact.

## Features

- Modern one-page React portfolio
- Architecture-oriented positioning (hero + about)
- Experience, education, skills, and certifications sections
- Blog preview section
- SEO metadata (title, description, Open Graph, Twitter)
- Structured data (`Person` JSON-LD) for better indexing
- Uses CV and provided images from `public/`

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
